# 珠免酒鬼人格测试 — 产品需求文档（PRD）

> 版本：1.0 | 日期：2026-05-13 | 状态：已上线

---

## 1. 项目概述

### 1.1 项目背景

珠海免税集团（珠免）希望借助趣味人格测试，提升用户参与度与品牌认知，同时收集用户对酒类产品的消费偏好数据，为后续精准营销和选品策略提供数据支撑。

### 1.2 产品定位

一款面向珠免消费者的 **MBTI 风格酒类人格测试 H5 应用**。用户完成 18 道趣味题目后，获得"你的喝酒人格是什么"结果（对应一款酒品），兼具娱乐性与导购价值。

### 1.3 目标用户

- 珠海免税商场/免税店的消费客群
- 对酒类产品感兴趣的年轻消费者（25-45 岁）
- 微信生态内的社交分享用户

## 2. 产品功能

### 2.1 用户端（问卷页面）

#### 2.1.1 问卷流程

| 步骤 | 说明 |
|------|------|
| ① 开始页 | 品牌 header（🍸 你的喝酒人格是什么？），引导用户进入测试 |
| ② 答题 | 18 道单选题，逐题滑动作答，实时记录选项 |
| ③ 计分 | 基于 4 个维度（E/I、S/N、T/F、J/P）累计分数 |
| ④ 生成结果 | 匹配 16 种人格 → 对应 16 款酒品，展示人格名称、tagline、详细描述 |
| ⑤ 分享/联系 | 结果页展示客服二维码，引导用户咨询或购买 |

#### 2.1.2 题目设计

共 **18 道题**，覆盖 4 个维度：

| 维度 | 含义 | 题目数 | 示例题目 |
|------|------|--------|----------|
| E / I | 聚场型 vs 独酌型 | 约 4-5 题 | "周五晚上终于来了，你最想怎么过？" |
| S / N | 经典型 vs 尝鲜型 | 约 4-5 题 | "看到一瓶没见过的酒，你的第一反应是？" |
| T / F | 实用型 vs 享受型 | 约 4-5 题 | "一瓶酒标价 800 块，你的第一反应是？" |
| J / P | 目标型 vs 随缘型 | 约 4-5 题 | "心心念念的酒断货了，你怎么办？" |

每题 2 个选项，每个选项对应一个维度的倾向。部分题目设有隐藏维度（hdim/hval），用于更细粒度的人格判定。

#### 2.1.4 维度分析

结果页展示 4 组维度得分对比（如 E: 7 vs I: 3），让用户直观了解自己在各维度的倾向程度。

#### 2.1.5 客服引导

结果页底部展示珠免客服二维码（图片），引导用户扫码咨询、购买推荐酒品。

### 2.2 管理端（数据看板）

#### 2.2.1 登录鉴权

- 用户名/密码登录（默认 admin / admin123，通过环境变量可配置）
- Token 机制鉴权，24h 自动过期
- 未登录无法访问任何数据接口

#### 2.2.2 汇总卡片

| 卡片 | 数据来源 | 说明 |
|------|----------|------|
| 📊 总参与人数 | submissions 表 COUNT | 累计参与人次 |
| 📅 今日新增 | 当日 COUNT | 含较昨日环比变化 |
| 🏆 最热人格 | mbti_type GROUP BY 排序 | 当前周期内人数最多的人格类型 |
| 📈 日均参与 | 总数 / 活跃天数 | 起止日期范围 |

#### 2.2.3 图表

| 图表 | 类型 | 数据 |
|------|------|------|
| 🥧 人格分布 | 饼图（环形） | 各 MBTI 类型占比 |
| 📊 维度倾向对比 | 柱状图 | E/I、S/N、T/F、J/P 八个维度总得分 |
| 📈 每日参与趋势 | 折线图（面积填充） | 按日期的参与人数变化 |

#### 2.2.4 筛选

- 日期范围（开始日期 ~ 结束日期）
- MBTI 类型筛选（下拉选择 16 种类型）
- 查询 / 重置按钮

#### 2.2.5 数据明细表格

| 字段 | 说明 |
|------|------|
| ID | 自增主键 |
| 时间 | 提交时间（本地化显示） |
| MBTI | 人格类型（带颜色 badge） |
| 人格名称 | 对应酒品名 |
| E/I | 维度得分 |
| S/N | 维度得分 |
| T/F | 维度得分 |
| J/P | 维度得分 |
| 设备 | 屏幕尺寸 |

支持分页（50 条/页）。

#### 2.2.6 备注模块

- 添加备注：文本输入框，回车或点击"发布"提交
- 备注列表：按时间倒序展示，显示内容和创建时间
- 删除备注：每条备注右侧"删除"按钮，需二次确认
- 实时更新：增删后自动刷新列表

#### 2.2.7 导出

- 一键导出 CSV 文件（含 BOM，Excel 可直接打开）
- 支持按当前筛选条件导出
- 导出字段：ID、提交时间、MBTI 类型、人格名称、8 维度得分、设备、屏幕

## 3. 技术架构

### 3.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | HTML + CSS + JavaScript | 无框架，纯原生实现 |
| 图表 | Plotly.js 2.27.0 | CDN 引入，交互式图表 |
| 后端 | Node.js + Express 4.18 | RESTful API |
| 数据库 | sql.js（SQLite 纯 JS 实现） | 无需原生编译，内存运行 + 定期持久化 |
| 部署 | Railway 免费套餐 | 海外服务器，零成本全托管 |

### 3.2 数据库设计

**submissions 表（问卷提交记录）**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| created_at | DATETIME | 提交时间 |
| mbti_type | TEXT | MBTI 类型（如 INFP） |
| type_name | TEXT | 人格名称/酒品名 |
| dim_e ~ dim_p | INTEGER | 8 个维度得分 |
| answers | TEXT | 原始答案 JSON |
| user_agent | TEXT | 浏览器 UA |
| screen | TEXT | 屏幕尺寸 |

**notes 表（备注）**

| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER PK | 自增主键 |
| content | TEXT | 备注内容 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### 3.3 API 接口

| 方法 | 路径 | 鉴权 | 说明 |
|------|------|------|------|
| POST | /api/submit | 无 | 提交问卷数据 |
| GET | /api/stats | ✅ | 获取统计图表数据 |
| GET | /api/data | ✅ | 获取明细数据（分页） |
| GET | /api/export | ✅ | 导出 CSV |
| POST | /api/login | 无 | 管理员登录 |
| POST | /api/logout | ✅ | 退出登录 |
| GET | /api/notes | ✅ | 获取备注列表 |
| POST | /api/notes | ✅ | 新增备注 |
| DELETE | /api/notes/:id | ✅ | 删除备注 |

### 3.4 部署架构

```
用户浏览器
    │
    ├── /survey.html  →  问卷页面
    └── /dashboard.html  →  数据看板
            │
            ▼
    Railway (Node.js + Express)
            │
            ├── sql.js (内存 SQLite)
            │       └── /tmp/survey.db（定期持久化）
            │
            └── 静态文件服务 (public/)
```

- **域名**：https://zm-quiz-production.up.railway.app
- **代码仓库**：https://github.com/Zzsomeone/zm-quiz
- **零成本**：Railway 免费套餐，无域名/服务器费用
- **数据持久化**：每 30 秒自动保存，进程退出时保存
